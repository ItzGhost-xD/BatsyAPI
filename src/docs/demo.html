🚀 Deploy Guide
1 — Clone & configure
git clone https://github.com/ItzGhost-xD/BatsyAPI
cd BatsyAPI
cp .env.example .env
# Fill in DISCORD_BOT_TOKEN, ETAG_SECRET, MONGODB_URI
2 — Docker (easiest)
docker compose up -d
# Open http://localhost:3000
3 — VPS with PM2
npm install
npm install -g pm2
pm2 start src/index.js --name batsy-api
pm2 save && pm2 startup
4 — Auto-deploy from GitHub
# On your server — run once:
cd ~/BatsyAPI
cat > deploy.sh << 'EOF'
#!/bin/bash
git pull origin main
npm install --omit=dev
pm2 restart batsy-api
EOF
chmod +x deploy.sh

# Then on GitHub → Settings → Webhooks:
# Payload URL: https://api.yourdomain.com/deploy
# (or use GitHub Actions — see README)
Discord Bot Setup
  1. Go to discord.com/developers/applications
  2. New Application → Bot → copy token
  3. Enable Privileged Intents:
    GUILD_MEMBERS + GUILD_PRESENCES
  4. Invite with bot scope, 0 permissions
  5. Paste token in .env

Features
  • Full presence parsing — online, idle, DND, offline, invisible
  • Platform detection — desktop, mobile, web
  • Spotify — track, artist, album art, progress
  • Rich Presence — games, VS Code, streaming
  • Custom status — emoji + text
  • Offline users — last known state preserved
  • Redis cache — ETag, Cache-Control, X-Cache
  • WebSocket — real-time subscriptions
  • HTML cards — embeddable presence card
  • MongoDB analytics — 30-day history
