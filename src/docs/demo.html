Discord Presence API

Real-time presence data for Discord users — status, Spotify, games, rich presence, custom status, platform detection, and more. REST + WebSocket. Redis-backed. Shardable.

View API Docs Try a Presence Card
What's included
🎯
Full Presence Parsing
Online, Idle, DND, Offline. Desktop / Mobile / Web platform flags per status.
🎵
Spotify & Rich Presence
Track, artist, album art, timestamps. Full rich presence for games and apps like VS Code.
⚡
Redis Cache
Shared fast cache across all shards. ETag + Cache-Control for HTTP clients. Sub-ms reads.
🔌
WebSocket Subscriptions
Subscribe to any user. Fan-out via Redis pub/sub so every node delivers updates.
📊
MongoDB Analytics
Every status change stored. Query history, status trends, and API usage stats.
🛡️
Rate Limiting & Auth
Per-IP limits on REST and WS. Optional API key gate with Bearer or x-api-key headers.
🌍
Horizontal Sharding
Run multiple bot instances with SHARD_ID / SHARD_COUNT. Keys namespaced per shard.
🎨
HTML Card + JSON
Browsers get a rich Discord-style card. Add ?format=json or use Accept header for JSON.
